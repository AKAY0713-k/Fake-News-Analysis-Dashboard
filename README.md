# Fake-News-Analysis-Dashboard
Interactive Power BI dashboard for analyzing fake news trends, platform performance, engagement, and misinformation patterns using dynamic visualizations.

## 📌 Overview
This project presents an interactive Power BI dashboard for analyzing fake and real news articles. It provides insights into misinformation trends, engagement, virality, and platform-wise distribution using dynamic visualizations.

## 🎯 Objectives
- Analyze fake and real news distribution.
- Identify platforms with the highest fake news rate.
- Compare engagement metrics.
- Discover trends over time.
- Provide actionable insights through interactive visualizations.

## 🛠️ Tools Used
- Power BI
- Python (Data Cleaning & Transformation)
- CSV Dataset


## 📈 Dashboard Features
- KPI Cards
- Fake vs Real News Distribution
- News Trend Analysis
- Platform-wise Fake News Rate
- Top Sources Analysis
- Interactive Filters (Category, Platform, Year, Country)

## 💡 Key Insights
- Around 49% of the analyzed articles were classified as fake news.
- Fake news spreads faster than real news on several platforms.
- Video and mobile platforms show higher fake news rates.
- The dashboard helps identify misinformation trends and high-risk sources.
